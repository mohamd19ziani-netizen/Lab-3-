ЛАБОРАТОРНАЯ РАБОТА №3

Тема: Поиск минимального количества ходов шахматного коня

Студент: Зиани Мохаммед

Группа: 090304-РПИа-о25

Цель работы

Изучить применение алгоритма поиска в ширину (BFS) для нахождения минимального количества ходов шахматного коня на шахматной доске.

Постановка задачи

Даны обозначения двух полей шахматной доски (например, A5 и C2). Необходимо определить минимальное количество ходов, которое требуется шахматному коню для перехода из первого поля во второе.

Описание решения

Для решения задачи используется алгоритм поиска в ширину (Breadth-First Search, BFS).

Шахматная доска рассматривается как граф размером 8×8. Каждая клетка является вершиной графа, а возможные ходы коня представляют собой ребра между вершинами.

Алгоритм BFS гарантирует нахождение кратчайшего пути в невзвешенном графе, поэтому он позволяет определить минимальное количество ходов коня.

Алгоритм работы программы

1. Считываются начальная и конечная клетки шахматной доски.
2. Координаты переводятся в числовой формат.
3. Создается очередь для реализации алгоритма BFS.
4. Последовательно рассматриваются все возможные ходы коня.
5. Для каждой новой клетки запоминается количество сделанных ходов.
6. После достижения конечной клетки выводится минимальное количество ходов.

Результат работы программы

Программа корректно определяет минимальное количество ходов шахматного коня между двумя заданными клетками шахматной доски.

Вывод

В ходе выполнения лабораторной работы был реализован алгоритм поиска в ширину (BFS) для нахождения минимального количества ходов шахматного коня. Полученная программа успешно решает поставленную задачу и демонстрирует эффективность применения алгоритмов обхода графов.
